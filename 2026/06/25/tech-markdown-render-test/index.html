16. 总结

如果这篇文章可以正常显示, 说明当前 Hexo 至少支持常见技术博客格式, 包括标题, 段落, 列表, 表格, 代码高亮, 引用, HTML 和 SVG 图表.

如果 Mermaid 没有被渲染成流程图, 这是正常现象. 后续可以按需安装插件来增强图表能力.
